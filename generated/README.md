# Photon Relay

Local browser-based science olympiad scenario learning game generated from `source/textbook.pdf`.

Open:

```text
generated/index.html
```

The game uses HTML, CSS, and Vanilla JavaScript only. Progress is saved in `localStorage`.

Spoiler-safe review:

```text
generated/data/public-review/spoiler-safe-report.md
```

Spoiler-heavy artifacts:

```text
generated/data/sealed/
```

Do not open sealed files unless you want story spoilers.
